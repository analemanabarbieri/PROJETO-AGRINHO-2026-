# Agrinho 2026 - Sustentabilidade e Tecnologia no Campo

## 🎯 Objetivo do Projeto
Este site foi desenvolvido para o Concurso Agrinho 2026, abordando o tema oficial: *"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"*. O objetivo é demonstrar de forma clara e interativa como ferramentas de Inteligência Artificial, sensores e práticas sustentáveis atuam lado a lado na proteção de biomas e no aumento da eficiência da agricultura familiar e corporativa.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout responsivo e transições fluidas.
* **JavaScript (Vanilla):** Lógica interativa da calculadora de CO₂ e manipulação dinâmica do menu e modo de acessibilidade.

## ⚙️ Instruções de Uso
1. Navegue pelas seções usando o menu superior dinâmico (adaptável para celulares).
2. Use o botão **Alto Contraste** no topo da página para ajustar a acessibilidade visual de cores.
3. Vá até a **Calculadora de Emissão de CO₂**, digite o consumo de combustível e clique em "Calcular Impacto" para ver os dados simulados baseados em tecnologias agrícolas reais de 2026.
